# ⏳ The 60-Minute JavaScript Technical Challenge

Welcome to the **Shopping Cart API Challenge**! This is a 1-hour, hands-on coding assessment designed to test your core JavaScript skills, logic, and comfort with DOM manipulation.

The project mimics a real-world development environment. You will be working on a clean file (`app.js`) to complete specific functions, and you'll have an automated test suite (`tests.js`) running behind the scenes to validate your code in real-time.

---

## 🎯 Topics Covered

This challenge evaluates 16 fundamental and modern JavaScript concepts, split into three main categories:

### 1. DOM Manipulation
* Changing text and HTML content
* Creating HTML elements dynamically
* Adding multiple elements efficiently to the DOM (Performance optimization)
* Removing elements from the DOM

### 2. Logic, Strings & Parameters
* Ternary operators
* Logical operators (`AND` / `OR`)
* Early returns pattern
* Template literals
* Object/Array destructuring
* Default function parameters

### 3. Advanced Objects & Arrays
* Enhanced object literals (Shorthand properties)
* Rest and Spread operators
* Useful array methods (e.g., `.filter()`)
* Array reduction using `.reduce()`
* Looping through and iterating over objects
* Returning objects with implicit return arrow functions

---

## 💻 How to Get Started & Run the Tests

You have 60 minutes to complete the challenges inside `app.js`. Your goal is to make all tests output `✅ PASSED`.

You can run this project directly in your browser without installing anything using **GitHub Codespaces**.

### Step-by-Step Instructions:

1. **Launch the Environment:**
   * On this GitHub repository page, click the green **"Code"** button in the top right.
   * Select the **"Codespaces"** tab.
   * Click **"Create codespace on main"**.
   * *A full web-based VS Code environment will load in a few seconds.*

2. **Write Your Code:**
   * Open the **`app.js`** file. This is the **only** file you need to modify.
   * Write your solutions inside the designated `// YOUR CODE HERE` blocks.

3. **Run the Test Suite (The Native Play Button):**
   * Go to the left sidebar and click on the **Run and Debug** icon (the play button with a small bug insect next to it).
   * At the top of that sidebar, make sure the dropdown menu says **"Run Challenge Tests"**.
   * Click the green **Play (Triangle)** button.
   * *The tests will run automatically in the integrated terminal below, showing you which tasks are `✅ PASSED` or `❌ FAILED`.*

---

*Alternative Local Setup: If you prefer to run it locally on your machine, clone this repository, open it in VS Code, open your terminal, and run `node tests.js`.*

**Good luck! Your 60-minute timer starts now! 🚀**
